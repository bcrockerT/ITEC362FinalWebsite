The basics of HTML

What is HTML?

HTML, or Hyper Text Markup Language, is coding language used to create webpages. The code is interpreted by your web browser, and converted into visual stuff that you can see. Yes.

This site, along with another I will soon recommend, will be all that you will need to get a basic grasp on the concepts needed to create a functional, somewhat stylish website of your own. w3schools is a great resource to keep in mind while making your own sites, and will be extremely valuable for any questions you might have.

The HTML skeleton

To begin our HTML journey, I'm going to introduce HTML to you the same way it was introduced to me. I was shown the basic layout for HTML, and my teacher called it the "HTML Skeleton". The name is quite fitting because no matter the complexity of your website, it will always begin in a state very similar to this. The skeleton will contain all the minimal elements you need to create a basic webpage.

					
<HTML>

	<head>

		<title>Hello World!</title>

	</head>

	<body>

		Hello World!
	</body>

					
</HTML>


HTML Basics

The following table contains the required elements to make a very basic webpage. No matter the webpage, it will contain these elements. A brief description will also be given, so that you at least understand what each piece does.

Basic HTML Tags
TagPurpose
HTML The <HTML> tag defines the start and end of your page. Everything will be within these tags.
header Content within the <HTML> tags will not be displayed on your website, but is used for things to describe your website. Things like meta data, links to external CSS, and the <title> are put here.
title This is where you put the title that will appear in the web browsers tab.
body The body contains all of the visible content of your webpage. Everything you want to share is within the <HTML> tags.
*This is not a complete list of HTML elements*

Other basic HTML elements

Now that we know all the required elements to create a webpage, lets cover some other tags that will let us change how our content will appear.

  • Header tags - <h1> - <h6>. As the name suggests, these tags are used for content headers. <h1> is the largest, with <h6> being the smallest
  • Lists
    • Ordered lists - the <ol> tag will create an ordered list, and by defualt uses numeric values to create an organized list
    • Unordered lists - the <ul> is used to create an unordered list, or one that uses bullet points.
    • List elements - the <li> tag follows the <ol> or <ul> tags, and is used to add things to your list.

HTML containers

Now that the basic ways to present content have been presented, lets move on to some of the ways to help organize. In HTML, there are many different kinds of containers that you can use to help organize, format, and position the content on your webpages. While most of these containers are functinally the same, they each serve their own purpose

  • nav - The <p> is used to store content relating to navigation on your site, like the bar at the top of this page that contains links to each page.
  • div - The <div> is a generic content container that can be used for almost anything, and is commonly used to separate content that will be styled a specific way.
  • main - The <main> is used to store the main content of your page.
  • aside - The <aside> is used for secondary content, maybe a sidebar or anything else located to the side of the main content
  • article - The <article> is used to contain independent articles or content on your page.
  • section - The <section> is for separating generic content.
  • header - The <header> is used for storing the content in your header, and is generally at the top of your page.
  • footer - The <footer> is used for footer information that goes at the bottom of your page.

As stated above, all of these containers are functionally the same, but in best practice they all have their uses in specific circumstances.
