Basics of CSS

CSS, or cascading style sheet is the term used to describe the file that controls all the styling found in a webpage. In most cases, a file that is separate from your HTML page will be used to help with the designing and styling of your web content. The CSS file doesn't add anything to your page, but rather formats and changes what already exists in your HTML file.

Types of CSS

  • Inline
  • Internal
  • External

Inline

Inline CSS is inputted within the HTML element itself, such as in an <h1> or a <p> tag. This kind of CSS is used in some situations where you may what a single element to appear unique or to help it stand out a little more from the rest of your content.

				
					
<span style="color:purple;">Example of inline CSS!>span>

Internal

Internal CSS forgoes the need for an external style sheet, and instead you create all of your styling declarations within the HTML file itself. For small scale projects, internal CSS may be much more convenient and easy to implement, however it is usually best to use external CSS. As you will soon see, creating internal CSS is very quick and easy.

					
<html>
<head>
<title>Example title</title>
<style>
/*This is purely an example, you can choose any elements and declarations you wish*/
 body{
	background-color:black;
	color:white;
}
  h1{
	font-size:20px;
}
</style>
</head>
<body>
</body>
</html>

External

As mentioned above, CSS can be saved in its own external file. There isn't really anything special about a .css file, its pretty much a text file that is used to hold all the styling information that will be used in your HTML page.

Considering this file is external, there has to be a way to link it to your HTML page. The code below shows how to connect your CSS and HTML files together, so that all that beautiful styling you've created can be seen.

					
<html>
<head>
<title>Example title</title>
<link href="style.css" rel="stylesheet">
</head>
<body>
</body>
</html>

General CSS formating and use

When it comes to CSS, there are 2 main things to keep in mind, the element, and the property in which you are going to modify.

In CSS, elements are the objects that you are trying to target. These elements can be any HTML tag that you have used, such as an <h1>, <div>, or even the <HTML> tag itself. No matter what element it is, you can style it in some way with the use of CSS.

Properties are the things that you are changing about an element, such as the color, font-size, or the padding/margins. Many different properties exist, much too many for me to list off here, so I will provide a link to the w3schools list of properties. Each property has specific valid values, and to know for sure its best to look up the property you are using if you are unsure.
