JavaScript Basics

What is JavaScript?

JavaScript is a scripting language that works in you web browser. JavaScript is basically the only programming language that you can use in you web browser, and it is very similar to Java, which is a completely different programming language. JavaScript is responsible for a lot of interactive, auto-updating websites that you may have seen. JavaScript is responsible for adding most interactivity you see in modern websites.

As stated above, JavaScript is similar to Java in that you can create fully functional scripts all in the web browser. In these scripts you can define variables, perform arithmetic, create loops, and make functions to perform more complex tasks. In this way, its comparee to Java, as they have similar syntax and can perform very similar tasks.

JavaScript can be performed both locally within the HTML file itself, or it can be saved and linked to from another file, much like CSS

				
					 <script src="myscripts.js"></script>

An extremely simple JS script

				
					<script>document.write('Hello World');</script>

Another very simple example of JS is the "Last Modified" timestampt in the footer of each of this sites pages. The script is very simple, it saves the time that the HTML document was last modified, saves it, then prints it.

				
					<script>
						var date = document.lastModified;
						document.write("Last Modified: "+date);
					</script>
